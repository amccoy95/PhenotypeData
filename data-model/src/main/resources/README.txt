mp-hp.owl is generated by hand whenever we want the latest monarch mp-hp associations.
See https://www.ebi.ac.uk/seqdb/confluence/display/MouseInformatics/Generate+mp-hp+hybrid+ontology for instructions on
how to build by hand. If you build mp-hp.owl by hand, please replace the file in this resources directory with the new
version.

EMAP-EMAPA.txt is generated by hand and is required for correct OntologyParserFactory operation. It live in impc_ontologies
along with the rest of the owl files.
